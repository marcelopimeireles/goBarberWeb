# GoBarber Web
## GoBarber Web client for Barber Shops
## built with React / Redux Saga

#### Main features
- React styled components
- Redux Actions and Reducers
- Redux Saga middlewares
- Routing thought Redux Saga
- Reactotron Debbuging
